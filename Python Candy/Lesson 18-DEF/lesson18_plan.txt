-Aşağıdakı nümunədə tək arqument (fname) olan funksiya var.

5)Əgər funksiyaya neçə ədəd dəyər verəcəyimizi bilmiriksə o zaman funksiyada 
arqumenti (*args) ilə işarələyirik
def my_function(*kids):
  print("The youngest child is " + kids[2])

my_function("Emil", "Tobias", "Linus")

6)Keyword Arguments -> burda dəyişən sırası önəmli deyil
def my_function(child3, child2, child1):
  print("The youngest child is " + child3)

my_function(child1 = "Emil", child2 = "Tobias", child3 = "Linus")

7)Arbitrary Keyword Arguments, **kwargs -> funksiyaya neçə arqument əlavə edəcəyinizi bilmiriksə
def my_function(**kid):
  print("His last name is " + kid["lname"])

my_function(fname = "Tobias", lname = "Refsnes")
